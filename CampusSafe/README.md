
 **Start the Application:**
   Run the server to view the application:
   ```bash
   python manage.py runserver 8080
   ```
 **Access the Site:**
   Open your browser and navigate to ``http://127.0.0.1:8080``.

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Django (Python)
